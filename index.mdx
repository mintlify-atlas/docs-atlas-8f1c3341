WebTransfer Lite is a lightweight Node.js server that lets you transfer files between devices on the same local network. Start the server, scan the QR code from your phone, and drag files back and forth — no cloud, no accounts, no installation on the receiving device.

## How it works

<Steps>
  <Step title="Install and initialize">
    Clone the repo, install dependencies, and run `pnpm run init` to generate a secure JWT secret in your `.env` file.
  </Step>
  <Step title="Start the server">
    Run `pnpm start`. The terminal displays your LAN URL, a one-time password, and a QR code.
  </Step>
  <Step title="Connect from any device">
    Open the URL on a PC and enter the password, or scan the QR code from a mobile device for instant access — no password required.
  </Step>
  <Step title="Transfer files">
    Upload files from your device, browse and download files from other devices. The session expires automatically when the configured time elapses.
  </Step>
</Steps>

## Key features

<CardGroup cols={2}>
  <Card title="JWT authentication" icon="shield-halved">
    Every session is secured with a signed JWT. Tokens expire automatically, and the server shuts down when the session ends.
  </Card>
  <Card title="QR code access" icon="qrcode">
    A QR code containing a pre-signed token is printed to the terminal. Scanning it grants instant mobile access.
  </Card>
  <Card title="Streaming uploads" icon="arrow-up-from-bracket">
    Files are streamed directly to disk — large uploads never saturate RAM regardless of file size.
  </Card>
  <Card title="Auto-shutdown" icon="clock">
    The server exits automatically when the session expires, freeing the port and leaving the machine unexposed.
  </Card>
</CardGroup>
